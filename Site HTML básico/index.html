Bem-vindo à Página Inicial!

Este é um site simples criado apenas com HTML. Ele contém 4 páginas: Inicial, Projetos, Artigos e Contato.

Use a barra de navegação acima para navegar entre as páginas.

Detalhes do site:
- Responsivo: Adapta-se a telas mobile (viewport meta e flexbox no nav).
- Navegação consistente: Mesma barra em todas as páginas com hover effects.
- Páginas estáticas: Apenas HTML, sem JS ou CSS externo (estilos inline).
- Idioma: Português (lang="pt-BR").
